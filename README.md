### Controls
| Action        | Key(s)               |
|---------------|----------------------|
| Move          | `W`, `S`, `A`, `D`   |
| Camera Lock (good for laptop)  | `CTRL`               |
| Camera Move   | `RMB` + `MOVE MOUSE` |
| Spawn Smoke  (good for laptop)  | `MMB /  G`           |
| Close         | `ESC`                |
| Change Smoke Settings      | `ImGui Menu`         |
